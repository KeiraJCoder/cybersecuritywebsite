Password Policy Documentation

Introduction

A strong password policy is vital in protecting your personal and organizational data. This document outlines the requirements set by the Cybersecurity Escape Room for passwords.

Password Requirements:

  1. Length:
    • Your password must be at least 8 characters long.
  2. Numbers:
    • At least one numeral (0-9) must be used in the password.
  3. Special Characters:
    • The password must contain at least one special character from the following: ! @ # $ % ^ & * ( ) _ + - = [ ] { } ; : ' " \ | , . < > / ?
  4. Case Sensitivity:
    • It's recommended to use both uppercase and lowercase letters in your password for added security.
